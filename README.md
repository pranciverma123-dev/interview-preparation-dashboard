# Interview Preparation Dashboard

A React-based dashboard for tracking interview preparation progress.

## Features

- Topic-wise interview questions
- Progress tracking
- Weak topic analysis
- Personalized study plan
- Revision reminders
- Mock interview section
- Dark/Light theme
- Interactive charts and analytics

## Tech Stack

- React.js
- Context API
- React Router
- Recharts
- Local Storage
- CSS

## Installation

```bash
npm install
npm start
```

- Authentication
- AI-based Interview Suggestions
