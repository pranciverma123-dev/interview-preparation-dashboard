#  PrepAce – Interview Preparation Dashboard

A modern React-based Interview Preparation Dashboard that helps students track their interview preparation journey, monitor progress, identify weak topics, and generate a personalized study plan.

---

##  Project Overview

Preparing for technical interviews often becomes difficult because students lose track of solved questions, weak topics, and revision schedules.

PrepAce solves this problem by providing a centralized dashboard where users can:

* Track solved interview questions
* Monitor overall preparation progress
* Identify weak topics automatically
* Generate a personalized study plan
* Schedule revisions
* Visualize progress through charts
